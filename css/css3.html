CSS Colors:

CSS Color Names:

In CSS, a color can be specified by using a predefined color name:

CSS Example:

Tomato

Orange

DodgerBlue

MediumSeaGreen

Gray

SlateBlue

Violet

LightGray


CSS Background Color:

You can set the background color for HTML elements:

CSS Example:

Hello World

This is a paragraph.


CSS Text Color:

You can set the color of text:

CSS Example:

Hello World

This is a blue coloured paragraph.

This is a green coloured paragraph.


CSS Border Color:

You can set the color of borders:

CSS Example:

CSS Color Values

In CSS, colors can also be specified using RGB values, HEX values, HSL values, RGBA values, and HSLA values.

CSS Example:

rgb(0, 255, 255)

#00ffff

hsl(180, 100%, 50%)
