How to start lampp server in ubuntu:

1)Open the terminal

2)Then add 'cd /opt'

3)Next open 'cd lampp'

4)Type the password of your computer, by adding 'sudo -i'

5)Repeat the first and second step by giving command 'cd /opt' and 'cd lampp'

6)Next give the command 'chmod -R 777 htdocs'

7)Open the file by the command 'ls'

8)Then start lampp server by giving the command './manager-linux-x64.run'
